# Level 1: 启蒙期知识点详解
**级别**: 🌱 Level 1 (0基础 → 30级)  
**目标**: 认识围棋，会下完整一盘棋  
**棋盘**: 9×9  
**对局数**: 约20盘

---

## 1.1 棋盘与规则认知

### 📚 知识点 1.1.1: 认识围棋

#### 基础概念
**什么是围棋？**
围棋起源于中国，已有4000多年历史，是世界上最古老的棋类游戏之一。围棋使用黑白两色棋子，在纵横各19条线（或其他规格）的棋盘上进行对弈。

**围棋的目标**
围棋的目标是围地。对局结束时，围住空点多的一方获胜。

**基本礼仪**
1. 对局前互相行礼
2. 黑棋先行（猜先决定）
3. 落子无悔
4. 对局结束互相行礼

#### 教学目标
- [ ] 了解围棋是两人对弈的围地游戏
- [ ] 认识棋盘、棋子
- [ ] 掌握基本对局礼仪
- [ ] 能正确摆放棋盘和棋子

#### 教学要点
- 用故事引入："四千年前，中国古人发明了围棋..."
- 展示棋盘："这是19×19的棋盘，有361个交叉点"
- 演示落子："棋子要放在交叉点上，不是格子里"

#### 常见错误
- ❌ 棋子放在格子里（不是交叉点）
- ❌ 一次拿多个棋子
- ❌ 移动已经落下的棋子

### 📚 知识点 1.1.2: 棋盘坐标

#### 基础概念
**棋盘结构**
- 标准棋盘：19条横线 × 19条纵线
- 交叉点：共361个
- 九星：棋盘上的9个黑点标记
- 天元：正中央的点

**棋盘分区**
- 角：四个角落区域
- 边：四条边的中间区域
- 腹（中央）：棋盘中央区域

**坐标表示**
- 横线：从上往下数第1-19线
- 纵线：从左往右数第1-19线
- 位置表示：如"4-4"表示第4条横线与第4条纵线的交点

#### 教学目标
- [ ] 能指出九星的位置
- [ ] 能找到天元
- [ ] 能区分角、边、腹
- [ ] 能说出指定位置的坐标

#### 教学要点
```
记忆口诀：
"四角四边一中央，
九星天元是标志，
三路线，四路线，
先从角上开始下。"
```

#### 练习题目
1. 请指出棋盘上的天元
2. 请指出4-4点的位置
3. 请指出3-3点的位置
4. 哪个位置是角？哪个是边？

### 📚 知识点 1.1.3: 落子规则

#### 基础概念
**基本规则**
1. **黑先白后**：黑棋先下第一手，然后双方轮流落子
2. **一次一手**：每次只能下一颗棋子
3. **落子无悔**：棋子落下后不能移动或悔棋
4. **落子位置**：棋子必须放在空着的交叉点上

**禁入点规则**
- 没有气的交叉点不能下子
- 特殊情况：可以提子时除外

#### 教学目标
- [ ] 理解黑先白后的规则
- [ ] 掌握轮流落子的顺序
- [ ] 知道落子无悔的道理
- [ ] 能正确放置棋子

#### 教学对话示例 (阿呆)
> 🐼 **阿呆**: "小朋友，围棋是两个人玩的游戏。你看，黑棋和白棋轮流下棋，就像你拍一我拍一一样。黑棋先下第一手，这是规矩哦~"
>
> 👶 **小朋友**: "那我可以一次下两颗吗？"
>
> 🐼 **阿呆**: "哈哈，不行哦！一次只能下一颗，就像一次只能吃一口饭一样。而且棋子放下去了就不能拿起来改哦，这叫'落子无悔'，是围棋的小规矩。"

#### 常见错误
| 错误行为 | 正确做法 |
|---------|---------|
| 一次拿多个棋子 | 一次只拿一颗棋子 |
| 放在格子里 | 放在交叉点上 |
| 悔棋/移动棋子 | 落子无悔 |
| 在有子的地方下棋 | 只能放在空点上 |

## 1.2 气的概念

### 📚 知识点 1.2.1: 什么是气

#### 基础概念
**气的定义**
"气"是棋子的生命线！一颗棋子的气，就是它直线相邻的空交叉点。

**气的数量**
- 在棋盘中央：有4口气
- 在棋盘边上：有3口气
- 在棋盘角上：有2口气

**气的可视化**
```
中央棋子4口气：
   ○
○  ●  ○
   ○

边线棋子3口气：
   ○
○  ●
   ○

角上棋子2口气：
○  ●
   ○
```

#### 教学目标
- [ ] 理解"气"的概念
- [ ] 能数出单个棋子的气
- [ ] 知道气的重要性
- [ ] 能用"空气口袋"的隐喻理解气

#### 教学对话示例 (阿呆)
> 🐼 **阿呆**: "小朋友，你知道棋子是怎么呼吸的吗？"
>
> 👶 **小朋友**: "棋子也会呼吸吗？"
>
> 🐼 **阿呆**: "对呀！棋子需要'气'才能活着。你看，这颗黑棋旁边有几个空点，这就是它的'气'，就像我们的呼吸一样。"
>
> 👶 **小朋友**: "那没有气了会怎样？"
>
> 🐼 **阿呆**: "没有气，棋子就会被吃掉啦！所以我们要保护好自己的气，同时堵住对方的气~"

#### 隐喻系统
- **气 = 空气口袋/呼吸空间**
- 教孩子："就像我们不能屏住呼吸太久一样，棋子也需要空气口袋才能活着"

---

### 📚 知识点 1.2.2: 气的计算

#### 基础概念
**如何数气**
1. 找到要数的棋子或棋块
2. 从棋子出发，看直线相邻的空点
3. 数出所有空点的数量

**连接棋子的气**
当棋子连在一起时，它们共享所有的气！

```
两个连接的黑棋：
   ○
○  ● — ●  ○
   ○
   ○

这两颗黑棋共有6口气（不是8口，因为中间不是空点）
```

#### 教学目标
- [ ] 掌握数气的方法
- [ ] 能计算连接棋子的气
- [ ] 能判断棋子是否危险（气少）

#### 练习题目
1. 数出中央一颗棋子的气：___口
2. 数出边上两颗连接棋子的气：___口
3. 数出角上三颗连接棋子的气：___口
4. 下面棋块有几口气？（图示）

#### 评估标准
- 能正确数出单颗棋子的气：中央4口、边上3口、角上2口
- 能正确数出连接棋子的共享气
- 能快速判断棋子是否处于危险状态（气≤2）

### 📚 知识点 1.2.3: 没有气的棋子

#### 基础概念
**禁入点规则**
- 如果某个交叉点没有气，就不能在那里下子
- 这叫做"禁入点"或"自杀"

**例外情况**
可以提子时除外！如果下子后能提走对方的棋子，这个点就不是禁入点。

```
示例：
白 ○ ● 
   ○ ○

黑棋可以在A位下子，因为下子后可以提走白棋！
```

#### 教学目标
- [ ] 理解禁入点规则
- [ ] 能判断哪里不能下子
- [ ] 理解"提子例外"的规则

#### 教学对话示例 (阿呆)
> 🐼 **阿呆**: "小朋友，有些地方是不能下棋的哦~"
>
> 👶 **小朋友**: "为什么不能下？"
>
> 🐼 **阿呆**: "你看，如果这里下子，这颗棋子就没有气了，会被立即吃掉。这就像是'自杀'，围棋里是不允许的。"
>
> 👶 **小朋友**: "那什么时候可以下呢？"
>
> 🐼 **阿呆**: "如果你下子后，能吃掉对方的棋子，那就可以下！这叫'以其人之道还治其人之身'~"

## 1.3 提子与吃子

### 📚 知识点 1.3.1: 提子规则

#### 基础概念
**什么是提子**
当一颗或一组棋子的所有气都被对方堵住时，这些棋子就被"提走"了。

**提子的步骤**
1. 确认对方棋块没有气了
2. 拿走这些棋子
3. 把被提的棋子放在棋盘旁边（用于计算胜负）

**提子的重要性**
- 提子可以救活自己的棋子
- 提子可以破坏对方的棋形
- 被提的子在最后计算时会影响胜负

#### 教学目标
- [ ] 理解提子的条件（没气才能提）
- [ ] 掌握提子的操作步骤
- [ ] 知道要保留被提的棋子

#### 教学对话示例 (阿呆)
> 🐼 **阿呆**: "小朋友，你看这颗白棋，它只有一口气了！"
>
> 👶 **小朋友**: "那怎么办？"
>
> 🐼 **阿呆**: "如果黑棋下在这最后一口气上，白棋就被'吃'掉了！我们要把白棋拿出来，放在棋盘旁边。这叫'提子'。"
>
> 🐶 **阿呆**: "提掉的棋子很重要哦，最后我们要数一下谁提的子多，这也是胜负的一部分~"

---

### 📚 知识点 1.3.2: 吃子游戏

#### 基础概念
**什么是吃子游戏**
吃子游戏是围棋的简化版，以吃子数量定胜负。

**规则**
1. 使用9×9棋盘
2. 先吃掉5子（或10子）的一方获胜
3. 不需要计算围地

**为什么要玩吃子游戏**
- 简单易懂，快速上手
- 培养"气"的概念
- 练习吃子技巧
- 体验围棋的乐趣

#### 教学目标
- [ ] 能完成一盘吃子游戏
- [ ] 理解吃子游戏的胜负规则
- [ ] 体验围棋的乐趣

#### 教学策略
1. **目标明确**: "我们今天要吃掉对方5颗棋子！"
2. **鼓励尝试**: "大胆去吃，不用担心输！"
3. **及时反馈**: "好！你吃掉了对方一颗子！"
4. **复盘总结**: "这盘你吃了3颗，下次争取吃5颗！"

#### 变体玩法
- **限时赛**: 5分钟内谁吃得多
- **数量赛**: 先吃到5子获胜
- **围地+吃子**: 既比吃子又比围地

### 📚 知识点 1.3.3: 逃跑与救子

#### 基础概念
**什么时候需要逃跑**
当自己的棋子气很少（通常≤2），面临被吃的危险时，需要逃跑。

**逃跑的方法**
1. **长**: 向有气的地方延伸
2. **尖**: 斜向逃跑，更灵活
3. **跳**: 跳一步，速度更快

**什么时候不能逃**
- 逃跑后会损失更多棋子
- 周围已经被包围，无处可逃
- 有更重要的棋需要救

#### 教学目标
- [ ] 能识别危险的棋子（气少）
- [ ] 掌握基本的逃跑方法
- [ ] 能判断是否应该逃跑

#### 教学对话示例 (小智)
> 🦊 **小智**: "嘿！你的棋子只有一口气了，快逃啊！"
>
> 👶 **小朋友**: "怎么逃？"
>
> 🦊 **小智**: "往有气的地方跑！看，这边有个空点，你'长'出去，就多了一口气！"
>
> 👶 **小朋友**: "那如果对方追呢？"
>
> 🦊 **小智**: "那就继续跑！但是记住，如果跑不掉，就不要勉强，救其他更重要的棋子！围棋里，有时候放弃也是一种智慧~"

## 1.4 连接概念

### 📚 知识点 1.4.1: 什么是连接

#### 基础概念
**连接的定义**
相邻的同色棋子连成一体，叫做"连接"。

**连接的好处**
1. **共享气**: 连接后，所有棋子共享气
2. **变强**: 连接的棋子更难被吃
3. **团结**: 力量集中，更容易围地

**不连接的风险**
- 棋子分散，容易被各个击破
- 每块棋都需要做活，压力大

#### 教学目标
- [ ] 理解连接的概念
- [ ] 知道连接的好处
- [ ] 能识别是否已连接

#### 隐喻系统
- **连接 = 手拉手/团结**
- 教孩子："就像好朋友手拉手一样，棋子连接在一起就更强大"

---

### 📚 知识点 1.4.2: 连接的方法

#### 基础概念
**粘 (连接)**
最直接的方法，走在相邻的点上。
```
● — ○ → ● — ●
```

**虎 (虎口连接)**
隔一路，形成虎口，更灵活。
```
●   ○ → ●   ●
  ●       ○
```

**双 (两个虎口)**
最坚固的连接方式。

#### 教学目标
- [ ] 掌握粘的方法
- [ ] 了解虎和双
- [ ] 能选择适当的连接方式

---

### 📚 知识点 1.4.3: 分断的威胁

#### 基础概念
**什么是断点**
己方棋子之间如果没有直接相连，中间的空点就是"断点"。

**被分断的后果**
- 棋子变弱
- 气变少
- 容易被吃

**如何保护断点**
- 及时连接（粘、虎、双）
- 补强薄弱环节

#### 教学目标
- [ ] 能识别断点
- [ ] 知道被分断的风险
- [ ] 能保护断点

## 1.5 9路盘实践

### 📚 知识点 1.5.1: 9路盘布局

#### 基础概念
**为什么要用9路盘**
- 棋盘小，游戏时间短
- 更容易理解局部战斗
- 快速获得成就感

**9路盘的策略**
- 角更重要（占角效率高）
- 边次之
- 中央价值相对较低

**常见开局**
1. 占角（4-4点或3-3点）
2. 守角或挂角
3. 向边发展

#### 教学目标
- [ ] 能完成一盘9路对局
- [ ] 知道9路盘的布局顺序
- [ ] 理解角的重要性

---

### 📚 知识点 1.5.2: 简单胜负计算

#### 基础概念
**如何计算胜负**
1. 填满双方围住的空点（或分区计数）
2. 加上提掉的棋子数
3. 黑棋需要贴目（通常6.5或7.5目）
4. 比较双方总点数

**简化版**（适合儿童）
只数围住的地，提掉的子算1目/子。

#### 教学目标
- [ ] 能大致判断胜负
- [ ] 了解数棋的基本方法
- [ ] 养成数棋礼仪

#### 数棋礼仪
1. 双方同意结束对局
2. 整理棋盘，拿出死棋
3. 一方数棋，另一方监督
4. 确认结果，互相行礼
