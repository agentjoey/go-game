## 3.1 进阶吃子技巧

### 📚 知识点 3.1.1: 枷吃

#### 基础概念
**什么是枷吃**
枷吃是一种不直接打吃，而是通过包围对方逃跑路线来吃掉对方棋子的技巧。"枷"字原意是枷锁，形象地描述了这种包围方式。

**枷吃的特点**
- 不直接打吃，而是罩住对方
- 利用对方逃跑路线进行包围
- 需要预判对方逃跑方向

**枷吃的形状**
```
枷吃形状示例：

    ●
● ○   → 黑棋在下方枷吃，白棋无法逃脱
    ●

或：

●   ●
  ○   → 黑棋在下方枷吃
```

**枷吃的时机**
- 对方棋子逃跑路线上有己方棋子配合
- 对方棋子气少，逃跑空间有限
- 能够预判对方逃跑方向

#### 教学目标
- [ ] 理解枷吃的概念
- [ ] 能识别枷吃机会
- [ ] 能预判对方逃跑路线
- [ ] 能制造枷吃局面

#### 教学对话示例 (小智)
> 🦊 **小智**: "这招比双打更阴险！叫'枷吃'！"
>
> 👶 **小朋友**: "为什么不直接打吃？"
>
> 🦊 **小智**: "因为直接打吃，对方会逃跑！枷吃是罩住它的逃跑路线，让它跑不掉！就像...就像用笼子罩住小鸟！"
>
> 👶 **小朋友**: "那怎么罩？"
>
> 🦊 **小智**: "看白棋往哪跑，提前在那里等着！这需要预判，就像下棋要算三步！"

### 📚 知识点 3.1.2: 扑与倒扑

#### 基础概念
**什么是扑**
扑是故意送吃一颗棋子，以达到破坏对方眼位或棋形的目的。这是一种牺牲战术。

**什么是倒扑**
倒扑是一种特殊的扑，送吃后对方提子，然后可以提回更多的对方棋子。

**扑的应用场景**
- 破坏对方眼位
- 制造对方棋形缺陷
- 为后续攻击做准备

**倒扑的形状**
```
倒扑示例：

● ○ ●
  ●

黑棋在○处扑，白棋提掉黑棋后：

●   ●
  ○
  ●

黑棋可以再提回白棋！
```

#### 教学目标
- [ ] 理解扑的概念
- [ ] 理解倒扑的概念
- [ ] 能识别扑的机会
- [ ] 能制造倒扑局面

#### 教学对话示例 (悠悠)
> 🐱 **悠悠**: "扑，是围棋中的'舍'。"
>
> 👶 **小朋友**: "为什么要舍？"
>
> 🐱 **悠悠": "舍小利，得大利。你看，这颗棋子虽然被吃，但破坏了对方的眼位。"
>
> 👶 **小朋友**: "那倒扑呢？"
>
> 🐱 **悠悠": "倒扑更高明，舍一子，吃回更多。这需要算计，需要远见。"

### 📚 知识点 3.1.3: 征子进阶

#### 基础概念
**征子方向判断**
征子时必须保持正确的追吃方向，一般是斜向45度角追吃。

**引征**
利用征子威胁在其他地方获利。当对方有一个棋子面临被征吃的威胁时，这个威胁本身就可以作为劫材或在其他地方获利的手段。

**反征**
当对方征吃己方棋子时，利用己方其他棋子进行反击，反而吃掉对方征子的棋子。

**征子的禁忌**
- 前方有己方棋子阻挡
- 对方有接应棋子
- 征子路上有对方可以借力的棋子

#### 教学目标
- [ ] 掌握征子方向判断
- [ ] 理解引征的概念
- [ ] 理解反征的概念
- [ ] 能判断征子是否成立

## 3.2 基础对杀

### 📚 知识点 3.2.1: 对杀概念

#### 基础概念
**什么是对杀**
对杀是双方棋子互相包围，通过比拼气的数量来决定生死的战斗。对杀是围棋中最激烈的战斗形式。

**对杀的关键**
- 气的数量决定生死
- 先紧对方的气
- 同时保护自己的气

**对杀的目标**
- 杀掉对方，自己活
- 或者双方达成双活

#### 教学目标
- [ ] 理解对杀的概念
- [ ] 知道对杀的关键是气的数量
- [ ] 能识别对杀局面

---

### 📚 知识点 3.2.2: 气的计算

#### 基础概念
**外气**
棋块外面的气，是对杀中最直接的气。

**公气**
双方共享的气，对杀时需要特别注意。

**眼气**
眼中的气，有眼的一方在对杀中占优势。

**对杀公式**
- 无眼对杀：比较双方外气数量
- 有眼对杀：眼位带来额外的气（眼气）

#### 教学目标
- [ ] 能区分外气、公气、眼气
- [ ] 掌握对杀时的气数计算
- [ ] 理解有眼杀瞎的原理

---

### 📚 知识点 3.2.3: 对杀技巧

#### 基础概念
**长气**
增加自己的气，通过长、尖、跳等方法扩展生存空间。

**紧气**
减少对方的气，通过扑、点等方法压缩对方空间。

**做眼**
在对杀中做出眼位，增加眼气。

**避免对杀大忌**
- 双方互打吃：形成"对杀双活"或"对杀双死"
- 盲目长气：忽略对方紧气

#### 教学目标
- [ ] 掌握长气的方法
- [ ] 掌握紧气的方法
- [ ] 能在对杀中做眼
- [ ] 避免对杀中的常见错误

## 3.3 基础定式

### 📚 知识点 3.3.1: 星位定式

#### 基础概念
**小飞挂**
最基本的挂角方式，走在小目位置，是小飞距离。

**一间高挂**
高位挂角，走在四线，更注重势力发展。

**点三三**
直接点角，是现代围棋流行的下法，AI常用。

**星位定式的选择**
- 小飞挂：平衡，最常用
- 一间高挂：取势
- 点三三：取地，简化局面

#### 教学目标
- [ ] 掌握小飞挂
- [ ] 了解一间高挂
- [ ] 了解点三三
- [ ] 能根据局面选择挂角方式

---

### 📚 知识点 3.3.2: 小目定式

#### 基础概念
**小飞挂**
最常用的小目挂角方式。

**一间高挂**
高位挂角，取势。

**大飞挂**
宽松挂角，避免复杂战斗。

**小目守角**
- 小飞守角：最常用
- 单关守角：更注重势力

#### 教学目标
- [ ] 掌握小飞挂和一间高挂
- [ ] 了解大飞挂
- [ ] 掌握小飞守角

---

### 📚 知识点 3.3.3: 定式后的进行

#### 基础概念
**定式不是结束**
定式只是局部战斗的开始，后续的发展更加重要。

**发展方向的选择**
- 向边发展
- 向中央发展
- 根据全局选择

**与周围棋子的配合**
- 与己方棋子的配合
- 与对方棋子的关系
- 厚势与实地的平衡

#### 教学目标
- [ ] 理解定式只是开始
- [ ] 能选择发展方向
- [ ] 能考虑与周围棋子的配合

## 3.4 简单布局

### 📚 知识点 3.4.1: 占角方法

#### 基础概念
**星位**
- 一手占角
- 势力型，发展快
- 现代围棋最常用

**小目**
- 实地与势力平衡
- 变化多
- 传统布局常用

**三三**
- 偏重实地
- 发展慢
- 现代AI常用

**高目/目外**
- 特殊场合使用
- 注重外势

#### 教学目标
- [ ] 了解各种占角方法
- [ ] 知道各自的优缺点
- [ ] 能根据喜好选择

---

### 📚 知识点 3.4.2: 守角与挂角

#### 基础概念
**守角**
- 保护角部实地
- 小目：小飞、单关
- 星位：小飞、大跳

**挂角**
- 破坏对方守角
- 选择：先守角还是先挂角
- 时机：对方守角后及时挂角

#### 教学目标
- [ ] 掌握基本守角方法
- [ ] 理解挂角的目的
- [ ] 能判断守角还是挂角

---

### 📚 知识点 3.4.3: 拆边

#### 基础概念
**什么是拆边**
从角向边发展，扩大地盘。

**拆边的间距**
- 一拆：距离最近，最坚实
- 二拆：距离适中，最常用
- 三拆：距离远，发展快但薄

**拆边的配合**
- 高低配合
- 与友军的配合
- 不要拆到对方厚势附近

#### 教学目标
- [ ] 理解拆边的概念
- [ ] 掌握一拆、二拆、三拆
- [ ] 能判断拆边的间距
