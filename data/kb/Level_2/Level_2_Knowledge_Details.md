## 2.1 吃子技巧基础

### 📚 知识点 2.1.1: 双打

#### 基础概念
**什么是双打**
双打是同时打吃对方两块棋的技巧。当对方两块棋都只剩两口气时，一手棋同时威胁两块棋，对方只能救其中一块。

**双打的特点**
- 必得其一：对方无法同时救两块棋
- 高效吃子：一手棋制造两个威胁
- 常见形状：对方棋子分散且气少

**双打的基本形状**
```
双打形状示例：

● ○       ●   ○
  ○   或    ● ○
●            

黑棋下在○处，同时打吃两颗白棋
```

#### 教学目标
- [ ] 理解双打的概念
- [ ] 能识别双打的机会
- [ ] 能制造双打局面
- [ ] 能防守对方的双打

#### 教学对话示例 (小智)
> 🦊 **小智**: "嘿！看这个局面，白棋两块棋都只有两口气！"
>
> 👶 **小朋友**: "那怎么办？"
>
> 🦊 **小智**: "下在这里！一手棋同时打吃两块！这叫'双打'！白棋只能救一块，另一块就被吃了！"
>
> 👶 **小朋友**: "那白棋不是很惨？"
>
> 🦊 **小智**: "哈哈，这就是围棋的狡猾之处！要时刻注意自己的棋有没有被打双吃的危险！"

#### 练习要点
1. 识别双打机会（两块棋都气少且位置合适）
2. 制造双打局面
3. 防守双打（及时补棋，避免两块棋同时被打吃）

### 📚 知识点 2.1.2: 门吃

#### 基础概念
**什么是门吃**
门吃是利用己方棋子形成"门"字形，堵住对方逃跑路线的吃子方法。

**门吃的特点**
- 关门打狗：堵住出口，对方无处可逃
- 边线更有效：在边线或角部门吃威力更大
- 预判性强：需要提前布置

**门吃的形状**
```
门吃形状：

● ○ ●
  ●

或边线门吃：

● ○
●
```

#### 教学目标
- [ ] 理解门吃的概念
- [ ] 能识别门吃机会
- [ ] 能制造门吃局面
- [ ] 能避免被门吃

#### 教学对话示例 (小智)
> 🦊 **小智**: "看好了，这招叫'门吃'！"
>
> 👶 **小朋友**: "为什么叫门吃？"
>
> 🦊 **小智**: "就像把对方关在门里！你看，白棋想逃跑，但是两边都被堵住了，就像被关在房间里出不来！"
>
> 👶 **小朋友**: "好厉害！"
>
> 🦊 **小智**: "门吃在边线最厉害，因为边线只有三个方向，更容易堵死！"

### 📚 知识点 2.1.3: 抱吃

#### 基础概念
**什么是抱吃**
抱吃是利用己方棋子从两侧包围对方棋子，像拥抱一样抱住对方使其无法逃跑。

**抱吃的特点**
- 拥抱式包围：两侧夹击
- 利用己方棋子：需要有己方棋子在旁配合
- 边角更有效：在边角限制对方逃跑方向

**抱吃的形状**
```
抱吃形状：

● ○ ●

或角上抱吃：

● ○
●
```

#### 教学目标
- [ ] 理解抱吃的概念
- [ ] 能识别抱吃机会
- [ ] 能利用己方棋子进行抱吃

### 📚 知识点 2.1.4: 征子基础

#### 基础概念
**什么是征子**
征子是一种连续打吃的技巧，通过一路斜向追吃，最终吃掉对方棋子。

**征子的要领**
1. **方向正确**：始终保持正确的追吃方向
2. **不撞墙**：前方不能有己方棋子阻挡
3. **连续打吃**：每一步都打吃，不给对方喘息机会

**征子的形状**
```
征子过程：

初始：    第一步：  第二步：  最终：

● ○       ● ○       ● ●       ● ●
  ? →       ●  →     ● ○  →     ● ●
                      ?          ○
                                
                                
                                ●

黑棋一直斜向追吃，最终吃掉白棋
```

#### 教学目标
- [ ] 理解征子的概念
- [ ] 掌握征子的方向
- [ ] 能判断征子是否成立
- [ ] 能进行简单征子

#### 教学对话示例 (小智)
> 🦊 **小智**: "来，教你一招厉害的——征子！"
>
> 👶 **小朋友**: "征子是什么？"
>
> 🦊 **小智**: "就是一直追着对方打吃，直到把它吃掉！就像猫捉老鼠一样！"
>
> 👶 **小朋友**: "怎么追？"
>
> 🦊 **小智**: "斜着追！但是要注意，前方不能有你的棋子挡路，不然就像追老鼠追到墙角被自己人堵住了！"

#### 征子的禁忌
- ❌ 前方有己方棋子：征子会失败
- ❌ 方向错误：对方会逃掉
- ❌ 中途停止：对方会连回

## 2.2 基础死活形

### 📚 知识点 2.2.1: 真眼与假眼

#### 基础概念
**什么是眼**
眼是棋子围成的空交叉点。眼是棋子生存的基地。

**真眼 vs 假眼**
- **真眼**：对方无法破坏的眼，眼角都在己方控制下
- **假眼**：对方可以破坏的眼，至少一个眼角被对方控制

**真假眼的判断**
```
真眼示例（3个眼角都在控制下）：

● ● ●
● ○ ●
● ● ●

假眼示例（1个眼角缺失）：

● ● 
● ○ ○
● ● 
```

#### 教学目标
- [ ] 理解眼的概念
- [ ] 能区分真眼和假眼
- [ ] 知道两个真眼即活棋

#### 教学对话示例 (悠悠)
> 🐱 **悠悠**: "小朋友，知道什么是眼吗？"
>
> 👶 **小朋友**: "是眼睛吗？"
>
> 🐱 **悠悠**: "哈哈，差不多！眼就是棋子围成的小房间。有两个'房间'（眼），棋就活了！"
>
> 👶 **小朋友**: "那假眼呢？"
>
> 🐱 **悠悠**: "假眼就像有漏洞的房间，对方可以进来破坏。要做真眼，必须把'门'（眼角）都关好！"

### 📚 知识点 2.2.2: 直二、直三、直四

#### 基础概念
**直二**
- 形状：两个直线排列的空点
- 结果：**死棋**，无法做活
- 原理：无法做成两个真眼

**直三**
- 形状：三个直线排列的空点
- 结果：**先手活，后手死**
- 原理：可以做成一个半眼，需要先手占据要点

**直四**
- 形状：四个直线排列的空点
- 结果：**无条件活**
- 原理：可以做成两个眼

```
直二（死）：    直三（先手活）：    直四（活）：

● ● ●         ● ● ●            ● ● ● ●
●   ●         ●     ●          ●       ●
● ● ●         ● ● ●            ● ● ● ●
```

#### 教学目标
- [ ] 记住直二、直三、直四的形状
- [ ] 知道各自的死活结果
- [ ] 能在实战中识别

#### 记忆口诀
```
直二死，直三半，直四活棋不用看。
先手直三可活棋，后手直三很难办。
```

### 📚 知识点 2.2.3: 做眼方法

#### 基础概念
**扩眼**
扩大眼位的空间，让对方难以破坏。

**保护眼角**
确保眼位的四个眼角都在己方控制下。

**扑与倒扑**
- **扑**：故意送吃，破坏对方眼位
- **倒扑**：对方提子后，再提回更多

#### 教学目标
- [ ] 掌握扩眼的方法
- [ ] 能保护眼角
- [ ] 了解扑的基本概念

## 2.3 连接与分断

### 📚 知识点 2.3.1: 连接的重要性

#### 基础概念
**为什么要连接**
- 连接的棋子共享气，气变多
- 连接的棋子更强大，不容易被吃
- 力量集中，更容易围地或攻击

**连接的原则**
- **能连则连**：避免棋子被分断
- **优先连接弱棋**：气少的棋优先连接
- **注意效率**：选择最简洁的连接方式

#### 教学目标
- [ ] 理解连接的价值
- [ ] 知道什么时候应该连接
- [ ] 能评估连接的效率

---

### 📚 知识点 2.3.2: 分断技巧

#### 基础概念
**什么是分断**
切断对方的连接，使对方棋子分散变弱。

**分断的方法**
- **冲**：直线切断
- **刺**：斜向切断
- **挖**：利用对方断点进行分断

**分断的时机**
- 对方形状有缺陷时
- 分断后能攻击对方弱棋时
- 分断后能获利时

#### 教学目标
- [ ] 理解分断的概念
- [ ] 掌握基本分断手法
- [ ] 能判断分断的时机

#### 教学对话示例 (小智)
> 🦊 **小智**: "看到对方的断点了吗？"
>
> 👶 **小朋友**: "哪里？"
>
> 🦊 **小智**: "这里！两颗白棋之间有空隙，这就是断点！我们下在这里，把它们分开！"
>
> 👶 **小朋友**: "分开有什么用？"
>
> 🦊 **小智**: "分开后它们就不能共享气了！每块棋都要独立做活，压力就大了！这就是分断的威力！"

## 2.4 13路盘围地

### 📚 知识点 2.4.1: 金角银边草肚皮

#### 基础概念
**金角**
- 角部最容易围地
- 两手棋可以围住一个角
- 效率最高

**银边**
- 边上次之
- 需要三手棋以上才能围住
- 效率中等

**草肚皮**
- 中央最难围地
- 需要大量棋子
- 效率最低

#### 围地效率对比
| 位置 | 围地效率 | 所需手数 |
|-----|---------|---------|
| 角 | ⭐⭐⭐⭐⭐ | 2手 |
| 边 | ⭐⭐⭐ | 3-4手 |
| 中央 | ⭐ | 5手+ |

#### 教学目标
- [ ] 理解金角银边草肚皮的概念
- [ ] 知道布局的优先顺序
- [ ] 能在实战中应用

#### 记忆口诀
```
金角银边草肚皮，
布局先占角和边。
中央虽然很威风，
围地效率却最低。
```

#### 教学对话示例 (悠悠)
> 🐱 **悠悠**: "小朋友，知道为什么高手都喜欢占角吗？"
>
> 👶 **小朋友**: "为什么？"
>
> 🐱 **悠悠**: "因为角最容易围地！你看，只需要两颗子，角部就围住了。但在中央，可能要五六颗子才能围住同样的地。"
>
> 👶 **小朋友**: "那中央就不重要了吗？"
>
> 🐱 **悠悠**: "也不是，中央能影响四面八方。但是一开始，我们还是先占角、再占边，最后才进入中央。这叫'金角银边草肚皮'！"

### 📚 知识点 2.4.2: 围地基础

#### 基础概念
**什么是地**
围住空点就是地。地多者胜。

**围地的方法**
- **拆边**：从角向边发展
- **跳**：快速展开
- **大飞**：更大范围的展开

**围地的效率**
- 用最少的棋子围最多的地
- 棋子之间要保持联系
- 不要过于靠近对方强棋

#### 教学目标
- [ ] 理解围地的概念
- [ ] 掌握基本围地方法
- [ ] 能比较围地效率

---

### 📚 知识点 2.4.3: 简单攻防

#### 基础概念
**攻**
- 阻止对方围地
- 打入对方阵地
- 分断对方棋子

**防**
- 保护自己的地
- 连接自己的棋子
- 补强薄弱环节

**攻防平衡**
- 不要只攻不守
- 不要只守不攻
- 根据局面灵活调整

#### 教学目标
- [ ] 理解攻防的概念
- [ ] 能进行简单攻防
- [ ] 有攻防意识
