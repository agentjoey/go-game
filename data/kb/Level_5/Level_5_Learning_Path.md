## 🎮 学习重点

### 持续提升方法

**打谱学习**
```
每天打谱1-2局职业棋手对局
重点学习:
- 布局思路
- 中盘战斗手法
- 官子收束技巧
```

**死活训练**
```
每天做死活题10-20道
重点练习:
- 金柜角
- 大猪嘴
- 复杂手筋
```

**复盘总结**
```
每盘对局后复盘:
- 找出关键胜负手
- 分析失误原因
- 总结改进方向
```
